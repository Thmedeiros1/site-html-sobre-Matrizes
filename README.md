# Projeto – Exercícios de Matrizes (HTML/CSS)
Descrição

Este projeto foi desenvolvido como atividade acadêmica no primeiro semestre da graduação, com o objetivo de aplicar conceitos básicos de HTML e CSS na construção de uma página web estática.

A página apresenta a resolução de exercícios envolvendo matrizes, incluindo operações como soma, subtração, multiplicação por escalar, determinantes e tipos de matrizes, utilizando imagens para representar os cálculos e resultados.

# Objetivo do Projeto

Praticar a estruturação de páginas web com HTML5

Aplicar estilização básica utilizando CSS

Organizar conteúdo matemático de forma visual e compreensível

Consolidar conhecimentos iniciais de desenvolvimento web

Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilização e layout

Imagens (.png) – Representação visual das matrizes e cálculos

# Como Executar o Projeto

Faça o download ou extraia os arquivos do projeto

Abra o arquivo atividade.html em qualquer navegador web moderno

O conteúdo será exibido localmente, sem necessidade de servidor

# Observações

Projeto de caráter educacional
